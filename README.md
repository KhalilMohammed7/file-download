# Download File

A responsive, dependency-free download page for `folder.rar`.

## Deploy to GitHub Pages

1. Push the contents of this directory to a GitHub repository.
2. Open **Settings → Pages** in the repository.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the branch containing these files and the **`/ (root)`** folder, then save.

GitHub Pages will publish `index.html`, and the download button will serve
`files/folder.rar` directly from the same site.
